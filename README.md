# Perioperative Anticoagulation Manager

Clinical decision support tool for managing anticoagulation in surgical patients.

## Features
- CHA₂DS₂-VASc calculator
- Cockcroft-Gault renal function
- Evidence-based periprocedural protocols
- 2024 AHA/ACC & 2022 CHEST guideline compliant

## Usage
Open `index.html` in any modern browser or deploy to Vercel.
